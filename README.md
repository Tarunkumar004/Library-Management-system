# Library-Management-system
Library Management System is a simple application designed to manage library operations such as adding books, managing users, issuing and returning books, and tracking availability. It helps automate record-keeping, reduce manual work, and improve efficiency for librarians and users.
Java Swing Desktop Application

A Library Management System built using Core Java and Java Swing, designed to manage books, students, and book-issue records efficiently through a desktop-based graphical user interface.

This project demonstrates Java fundamentals, GUI programming, and basic application design, making it suitable for Java Intern / Fresher roles.
